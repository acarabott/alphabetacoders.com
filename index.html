Your goal

To become a l33t codeNinja, by learning Processing

Why Processing?

  • "Processing offers a way to learn programming through creating interactive graphics." - Getting Started with Processing by Casey Reas and Ben Fry (creators of Processing)

    "Interactive graphics" is a catch all term for games / art / visualisations.

    Later in the year we will start integrating sound into your sketches.

  • With Processing you will be able to write apps that run on:

    • Mac OS X
    • Windows
    • Linux
    • Android
    • Browsers (via processing.js)
    • iOS (via processing.js)

  • There are thousands of Processing examples online for you to learn from.

How you will be trained by your Sensei

We will be working through the examples from Getting Started with Processing in an interactive manner, discussing our work as we go.

To become a truly strong codeNinja you should experiment with the examples, breaking and re-inventing them!

After each session the examples will be written up and made available here so you can review them later.
